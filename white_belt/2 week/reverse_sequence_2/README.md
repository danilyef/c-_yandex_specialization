### Programming Task: Sequence Reversal - 2 

Implement *vector<int> Reversed(const vector<int>& v)* to return a copy of v with the numbers reversed.

### Example 

|           Vector v             |            Result              |
|:------------------------------:|:------------------------------:|
| 1, 5, 3, 4, 2                  | 2, 4, 3, 5, 1                  |
